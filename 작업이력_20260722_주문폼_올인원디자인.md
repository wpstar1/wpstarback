## 작업 A — 주문 폼 개선
요청: 주문 시 "키워드 + 하고싶은말" 입력 + 텔레그램 알림 포함.

원인: 키워드란은 원래 있었으나 도메인 권한 상승 상품에서 숨김 처리돼 있었고, "하고싶은말" 필드는 아예 없었음.

변경:
1. `index.html` 주문폼 — 이메일 아래 **하고싶은 말/요청사항 textarea(#message, 선택)** 추가
2. `script.js` `setupFormFields()` — 도메인 상품도 **키워드란 노출(선택 입력)** 로 변경 (기존: 숨김)
3. `script.js` `showPaymentInfo()` currentOrderData 에 `message` 수집 추가
4. `script.js` `submitOrder()` orderData 에 `message` 추가
5. 텔레그램 메시지 2곳(`sendTelegramPaymentNotification`, `sendTelegramViaServer` 콘솔/알림)에 `• 하고싶은 말` 추가
6. `server.js` `createOrderTelegramMessage()` 에도 하고싶은 말 조건부 추가(백엔드 일관성용)
